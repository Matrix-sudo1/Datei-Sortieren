This file contains various photo-related file extensions.

- .jpg
- .jpeg
- .png
- .gif
- .bmp
- .tiff
- .heif
- .webp
- .raw

## Camera Raw Formats
- .cr2
- .nef
- .arw
- .orf

## Others
